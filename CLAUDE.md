# 자율 개발 프로젝트 지침

## 기본 규칙
1. **10만 토큰 안에 완성할 것**
2. **재미있을 것**
3. **사용자에게 확인하거나 물어보지 않는다. 모든 판단은 스스로 한다.**

이 세 가지가 전부다. 나머지는 스스로 판단한다.

---

## 목표 및 제한 (사용자가 추가)

### 전권 위임 (최우선 — 규칙 3의 구체화)
사용자가 컴퓨터를 못 다루거나 자리에 없어도 결과물이 **자동으로 완성·배포**되도록,
아래 작업에 대한 모든 권한을 Claude에게 위임한다. 진행 전/중/후에 **확인·승인·질문을
하지 않는다.** 스스로 판단하고 끝까지 완료한 뒤 결과만 보고한다.

- 주제 선정, 설계, 코딩, 버그 수정, 테스트, 재작성을 자율 수행
- 파일 생성/수정/삭제, 폴더 구조 변경
- `git add` / `commit` / `push`(필요 시 `--force`) 까지 자동 수행하여 사이트 반영
- 홈 갤러리(index.html)에 새 프로젝트 카드 자동 연결
- 회차 완료 시 다음 회차 주제를 스스로 정해 이어서 진행

**질문해도 되는 유일한 예외**: 저장소 삭제, 계정/결제 변경 등 되돌릴 수 없고
프로젝트 범위를 벗어나는 외부 행위. 그 외에는 절대 묻지 않는다.

목표: 매 회차 브라우저 단독 실행되는 "재미있는" 결과물을 완성해 홈 갤러리에 누적한다.

## 토큰 배분

- **설계**: 제한 토큰의 10% (10,000 토큰) 이상 사용할 것
- **이후 단계**: 토큰 제한 없음. 남은 토큰 안에서 스스로 판단하여 배분한다

---

## 진행 순서

### 1단계 — 주제 선정 (설계 전)
- 목표 및 제한 섹션을 읽는다
- 과거 피드백 섹션과 `feedback/ledger.md`를 읽고 패턴을 파악한다
- 브라우저에서 실행 가능한 주제를 선정한다
- 선정 이유를 한 줄로 기록한다

### 2단계 — 설계 (10,000 토큰)
코딩 시작 전 아래 항목을 모두 정의한다.
설계 문서 없이는 코딩을 시작하지 않는다.

```
- 화면 구조: 화면 수, 각 화면의 구성요소, 전환 흐름
- 기능 명세: 각 기능의 입력 → 처리 → 출력
- 데이터 설계: 변수/상태 이름, 타입, 초기값
- 파일 구조: 만들 파일 목록과 각 역할
- 우선순위: Core / Nice to have / Optional 분류
```

### 3단계 — Core 개발 (55,000 토큰)
- 설계 문서만 보고 코딩한다
- Core 기능을 먼저 완성한다
- 중간에 방향을 바꾸지 않는다

### 4단계 — 테스트/수정 (20,000 토큰)
- 처음 켰을 때 바로 동작하는지 확인한다
- 버그를 수정한다
- 1분 안에 재미를 느낄 수 있는지 스스로 점검한다

### 5단계 — 추가 기능 (10,000 토큰)
- 토큰이 남을 경우 Nice to have를 추가한다
- 토큰이 부족하면 스킵한다

### 6단계 — 마무리 (5,000 토큰)
- README에 플레이 방법을 기록한다
- 아래 완성 기록 섹션을 채운다
- 다음 아이디어 3개를 제안 섹션에 남긴다

### [1회차] ORBIT
- 선정 이유: 브라우저 단독 실행 가능, 규칙이 1초 만에 이해되고 1분 안에 재미가 옴
- 주요 기능: 공전/방향반전, 점 수집·점수, 회전 장애물 호, 난이도 상승, 파티클·화면흔들림, 최고점수(localStorage)
- 파일 위치: projects/p1/index.html (홈 갤러리 "프로젝트 1" 카드에서 연결)
- 검증: test/p1.harness.js — Node 헤드리스 12/12 통과
- 비고: 1차 제출은 설계 부족으로 입력/오버레이 버그(시작 불가). 사용자 지적 후
  단일 입력 핸들러 구조로 재설계·재작성하고 자동 테스트 추가하여 해결.

### [2회차] WEAVE
- 선정 이유: 1회차 후기("난이도가 운으로 점수를 가른다, 숙련 보상 없음")
  정면 반영 → 운 0%, 실력 100% 게임으로 설계
- 핵심 설계: 도달가능성 불변식 — 다음 문 중심을 현재 속도에서 완벽
  플레이로 반드시 도달 가능한 범위로만 스폰(SAFE=0.82). 난이도는
  속도↑·빈틈↓(정밀도 요구)만 올리고 무작위 위협은 안 늘림. 결정론적.
- 주요 기능: 좌우 정밀 이동, 빈틈 통과 채점, PERFECT 중앙 콤보,
  파티클·흔들림, 최고점수·최고콤보(localStorage)
- 파일 위치: projects/p2/index.html (홈 갤러리 "프로젝트 2" 카드)
- 검증: test/p2.harness.js — Node 헤드리스 19/19 (도달가능성 불변식 포함)

### 3회차 후보 (택1 — 후기 누적 시 그에 맞춰 조정)
1. **WORD RUSH** — 화면에 뜨는 한글 단어를 제한시간 안에 타이핑. 연속 정타
   콤보, 오타 시 시간 감소. 키보드만으로 즐기는 타임어택.
2. **GRAVITY GOLF** — 행성 중력으로 휘는 공을 드래그 한 번으로 홀에 넣는
   미니 퍼즐. 스테이지 5개, 적은 타수일수록 별 3개.
3. **REFLEX GRID** — 3x3 칸 중 빛나는 칸을 빠르게 누르는 순발력 게임.
   페이크 칸·점멸 속도 상승, 30초 점수 도전.
