# 프로젝트 4 — PARRY

타이밍 방어 게임. 단독 `index.html` 하나로 브라우저에서 바로 실행됩니다.
(4회차 — 누적 후기 원칙 전부 계승)

## 누적 후기 반영

- (1·2회차) **실력 ~95%·운 ≤5%**: 모든 타격은 방향과 windup이 미리
  보이고, windup·간격을 가드 회전시간 이상으로 묶어 완벽 플레이면
  **반드시 막을 수 있음**. 무작위는 업그레이드 제시·보너스 샤드(≤5%)만,
  회피 불가능한 죽음 없음.
- (2회차) **구조/기믹**: 스테이지 / ⚔중간보스 / 👑보스(HP·클리어 +15) /
  스테이지 사이 무작위 3종 강화 + 보호막 / 샤드.
- (3회차) **어지러움 완화**: ECHO의 끊임없는 회전 대신 **불연속·차분한
  화면**(가드 + 들어오는 타격 마커), 흔들림 약화. **선택지 단조 해소**:
  8종 풀에서 매번 무작위 3종. **난이도 급화**: windup·간격·패링각 기울기↑.

## 플레이 방법

1. `index.html` 더블클릭 (또는 `https://viennaru-prg.github.io/projects/p4/`)
2. 화면 **탭 / 스페이스 / Enter** 로 시작
3. **마우스·손가락**으로 막을 방향 지정 (키보드 **WASD·화살표**도 가능)
4. 가장자리에서 들어오는 마커가 **링에 닿는 순간** 그 방향을 막고 있으면 패링
5. 정확히/타이밍 좋게 막으면 PERFECT(점수·콤보 ↑), 샤드 획득
6. 6웨이브마다 스테이지업 → **무작위 3종 강화** 선택
7. 3의 배수 ⚔중간보스, 6의 배수 👑보스(HP 바, 격파 +15)
8. 빗맞으면 게임 오버(보호막 있으면 1회 무효). 0.7초 뒤 재시작
9. 최고 점수 자동 저장
