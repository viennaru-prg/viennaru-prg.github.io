# 프로젝트 6 — ORBITAL

결정론적 중력 궤도 조준 게임. 단독 `index.html` 하나로 실행됩니다.
(6회차 — 토큰 한도 15만으로 상향, "코어 메커니즘을 실제로 다르게" 반영)

## 무엇이 달라졌나 (5회차 피드백)

피드백: 1~5회차가 결국 "링/위협 회피"라는 같은 계열이라 아쉬웠다.
좋았던 건 죽어도 누적되는 보상(메타) 시스템.

→ **코어 메커니즘 전환**: 실시간 회피가 전혀 아님. 혜성을 **드래그로
조준**(방향·세기)해 발사하면 행성 중력장이 궤도를 휘게 만들고, 그
궤도로 **포탈에 명중**시키는 결정론적 계획·정밀 장르.
- **같은 조준 = 항상 같은 결과** → 운이 0인 순수 계산·실력. 점선
  예측 경로를 보고 계획(공정한 텔레그래프).
- 좋았던 **메타 진행 계승·발전**: 모은 별이 누적되어 **승천** →
  다음 항해는 추가 연료·긴 예측선으로 시작(영구 저장).

## 콘텐츠 (볼륨/감동)

- 5개 영역(테마·플레이버·사운드), 네임드 보스(다중 명중 + 회전 가디언),
  스테이지별 별/파(par) 채점, 무작위 3종 강화(8풀), 절차적 사운드·연출.

## 플레이 방법

1. `index.html` 더블클릭 (또는 `https://viennaru-prg.github.io/projects/p6/`)
2. 탭/스페이스로 시작
3. 시작 패드 근처에서 **드래그**: 방향 = 조준, 끄는 거리 = 세기.
   놓으면 발사. (키보드: ← → 각도, ↑ ↓ 세기, Space 발사)
4. **점선 예측 경로**를 보고 중력을 계산해 **초록 포탈**에 명중
5. 빨간 위험체·행성 코어·화면 밖 = 발사 실패(발사 횟수 소모)
6. 보스: 회전 가디언을 피해 여러 번 명중
7. 스테이지 클리어 시 별 획득 → 무작위 3종 강화
8. 발사를 모두 소진하면 항해 종료. 모은 별은 누적되어 승천
9. 0.7초 뒤 재시작. 최고 점수·승천 영구 저장

## 검증

`node test/p6.harness.js` — 결정론(같은 입력=같은 결과), **모든
스테이지(노멀·미니·보스·심층 30)가 브루트 솔버로 풀이 가능함을 증명**
(공정성=실력, 운 없음), 난이도 스케일·하한, 메타 저장/적용, 실게임
루프(풀이샷이 실제 클리어), 발사 소진→종료·메타 적립, 무작위 풀,
재시작. (35/35 통과)
