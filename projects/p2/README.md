# 프로젝트 2 — WEAVE

정밀 컨트롤 게임. 단독 `index.html` 하나로 브라우저에서 바로 실행됩니다.

## 만든 의도 (1회차 후기 반영)

ORBIT 후기: "난이도 상승이 컨트롤이 아니라 운으로 점수를 가르고, 실력으로
나아진다는 느낌이 없다." → WEAVE는 정반대로 설계:

- **운 없음.** 모든 문이 미리 보이고, 다음 문은 현재 속도에서 완벽히
  움직이면 **반드시 통과 가능**하도록 스폰을 제약(도달가능성 불변식).
- 죽음은 항상 "내가 못 움직였다" — RNG가 아니라 내 컨트롤 탓.
- 난이도 상승 = 무작위 위협 증가가 아니라 **속도↑ · 빈틈↓** 로
  요구 정밀도만 올라감. 연습하면 분명히 더 멀리 간다(숙련 곡선).
- 가운데로 정확히 통과하면 PERFECT 콤보.

## 플레이 방법

1. `index.html` 더블클릭 (또는 `https://viennaru-prg.github.io/projects/p2/`)
2. 화면 **탭 / 스페이스 / Enter** 로 시작
3. **마우스·손가락**으로 좌우 이동 (키보드 **← →** 도 가능)
4. 내려오는 문의 빈틈을 통과 → 점수. 빈틈 밖이면 게임 오버
5. 중앙에 가깝게 통과하면 PERFECT 콤보 누적
6. 점수가 오를수록 더 빠르고 빈틈이 좁아짐(운 요소는 없음)
7. 게임 오버 약 0.7초 뒤 다시 입력하면 재시작
8. 최고 점수·최고 PERFECT는 자동 저장
