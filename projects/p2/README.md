# 프로젝트 2 — PIXEL FALL

좌우 이동 코인 수집 게임. 단독 `index.html` 하나로 브라우저에서 바로 실행됩니다.

## 플레이 방법

1. `index.html` 더블클릭 (또는 `https://viennaru-prg.github.io/projects/p2/`)
2. 화면 **탭 / 스페이스 / Enter** 로 시작
3. **마우스·손가락**으로 좌우 이동 (키보드 **← →** 도 가능)
4. **노란 코인** = 점수 (연속으로 먹으면 콤보가 쌓여 점수 배율 ↑)
5. **빨간 폭탄**에 닿으면 게임 오버
6. 코인을 놓치면 콤보 리셋, 점수가 오를수록 레벨이 올라 더 빨라지고 폭탄이 늘어남
7. 게임 오버 후 약 0.7초 뒤 다시 입력하면 재시작
8. 최고 점수·최고 콤보는 브라우저에 자동 저장

## 검증

`node test/p2.harness.js` — 로드 예외, 화면 전환, 400프레임 무예외,
코인 수집(점수·콤보), 코인 놓침(콤보 리셋), 폭탄 충돌→게임오버,
재시작 디바운스, 최고점수 저장, 난이도 상승을 자동 단언. (15/15 통과)
