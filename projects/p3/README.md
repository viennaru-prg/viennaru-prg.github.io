# 프로젝트 3 — ECHO

원형 아레나 정밀 생존 게임. 단독 `index.html` 하나로 브라우저에서
바로 실행됩니다. (3회차 — 누적 후기 원칙을 모두 계승, ~50k 토큰 규모)

## 누적 후기 반영

- (1회차 ORBIT) **운 0%·실력으로만**: 모든 고리는 미리 보이고, 고리
  속도·틈 회전을 플레이어 능력 미만으로 수학적으로 묶어 완벽 플레이면
  반드시 생존 가능. 죽으면 100% 내 컨트롤 탓.
- (2회차 WEAVE) **단조로움 금지·구조/기믹**: 스테이지 진행, ⚔중간보스,
  👑보스(HP 바·클리어 +15 보너스), 스테이지 사이 **업그레이드 선택**
  (무작위 아님 — 전략 선택), 샤드 수집.
- 검증으로 "예고 정보만 쓰는 완벽 회피 AI가 최고난도·보스까지 무피격
  생존"을 단언 → 실력 게임임을 증명.

### 3회차 후기 반영 (개정)

- **어지러움 완화**: 초반 링 빈도↓(차분), 임박 링만 또렷·먼 링은 흐리게,
  화면 흔들림 약화.
- **단조로움 해소**: 업그레이드 8종 풀에서 **매번 무작위 3종** 제시(런마다
  다름) + **보호막**(피격 1회 무효) 신규.
- **난이도 급화**: 속도·틈·간격 기울기 대폭 강화로 후반이 확실히 더 빡셈.
- **운 ~5%**: 전역 방침대로 약간의 무작위(업그레이드 제시·보너스 샤드)만,
  회피 불가능한 죽음은 없음(완벽 회피 AI 생존 테스트로 계속 보증).

## 플레이 방법

1. `index.html` 더블클릭 (또는 `https://viennaru-prg.github.io/projects/p3/`)
2. 화면 **탭 / 스페이스 / Enter** 로 시작
3. **마우스·손가락**으로 이동 (키보드 **WASD·화살표**도 가능)
4. 좁혀오는 고리의 **초록 테두리 = 안전한 틈**. 그 각도 안에 있어야 생존
5. 고리를 통과할 때마다 점수·샤드 획득
6. 6웨이브마다 스테이지업 → **업그레이드 선택**(샤드 소비)
7. 3의 배수 스테이지 ⚔중간보스, 6의 배수 👑보스(HP 바, 격파 시 +15)
8. 단 한 번이라도 틈 밖에서 고리에 닿으면 게임 오버
9. 게임 오버 약 0.7초 뒤 다시 입력하면 재시작. 최고 점수 자동 저장
