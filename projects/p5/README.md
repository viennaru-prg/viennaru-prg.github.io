# 프로젝트 5 — AURORA

대형 여정형 생존 게임. 단독 `index.html` 하나로 실행됩니다.
(5회차 — "볼륨·재미·감동 상향" 방침 반영, 누적 원칙 전부 계승)

## 무엇이 커졌나 (피드백: 볼륨이 작다 / 재미·감동 부족)

- **여러 영역(Act)**: 여명의 평원 → 보랏빛 협곡 → 불꽃의 회랑 → 심연의
  정원 → 별의 끝. 영역마다 색·배경·플레이버 문구·사운드 모티프가 바뀜.
- **네임드 보스**: 루멘·녹스·이그니스… 보스마다 등장 연출·HP 바·격파
  보너스. (6의 배수 스테이지)
- **메타 진행 (재플레이 동기)**: 살아남아 모은 빛은 사라지지 않고
  **누적**되어 **승천(Ascension)** 레벨이 오름. 다음 런은 더 빠르고
  수호막을 지닌 채 시작. (localStorage 영구 저장)
- **빛조각 수집**: 아레나를 떠도는 오브를 모아 추가 점수·빛(순수 보너스).
- **연출/사운드**: 절차적 WebAudio 효과음·영역 전환음, 화면 플래시·
  영역별 배경 그라데이션·배너 연출. (사운드는 미지원 환경에서 자동 무시)

## 계승한 원칙

- **실력 ~95% / 운 ≤5%**: 모든 빛고리는 예고되고, 속도·틈 회전·간격을
  플레이어 능력 미만으로 묶어 완벽 플레이면 반드시 생존. 무작위는
  업그레이드 제시·보너스(≤5%)에만 — 회피 불가능한 죽음 없음.
- **구조/변주·난이도 급화·차분한 가독성**(임박 고리만 또렷, 흔들림 약화).

## 플레이 방법

1. `index.html` 더블클릭 (또는 `https://viennaru-prg.github.io/projects/p5/`)
2. 탭/스페이스로 시작, **마우스·손가락/WASD·화살표**로 이동
3. 좁혀오는 빛고리의 **초록 틈**으로 빠져나가기 (닿으면 게임 오버,
   수호막 있으면 1회 무효)
4. 떠도는 **초록 오브** 수집 = 보너스 점수·빛조각
5. 6웨이브마다 스테이지업 → **무작위 3종 강화** 선택
6. 3의 배수 ⚔중간보스, 6의 배수 👑네임드 보스(HP 바, 격파 +20)
7. 게임 종료 시 모은 빛이 **누적**되어 승천 → 다음 런 강화
8. 0.7초 뒤 재시작, 최고 점수·승천 영구 저장
