# 프로젝트 1 — ORBIT

원버튼 반응 게임. 단독 `index.html` 하나로 브라우저에서 바로 실행됩니다.

## 플레이 방법

1. `index.html` 더블클릭(또는 `https://viennaru-prg.github.io/projects/p1/`)
2. 화면을 **탭 / 클릭 / 스페이스 / ↑ / Enter** 하면 시작
3. 흰 공이 원을 따라 공전합니다. 입력할 때마다 **회전 방향이 반전**됩니다
4. **파란 점**에 도달하면 +1점 (속도가 조금씩 빨라짐)
5. **빨간 호**에 닿으면 게임 오버
6. 점수가 오를수록 빨간 호가 늘고 넓어집니다
7. 게임 오버 후 잠시 뒤(약 0.7초) 다시 입력하면 재시작
8. 최고 점수는 브라우저에 자동 저장됩니다

## 검증

`node test/p1.harness.js` — DOM/canvas를 스텁으로 만들어 로드 예외,
화면 전환, 3000프레임 무예외, 점수 획득, 충돌→게임오버, 재시작 디바운스,
최고점수 저장을 자동 단언합니다. (12/12 통과)
